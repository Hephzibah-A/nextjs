# Test Project

This repository is a test project created for practicing Next.js development. It contains sample code and experiments to help improve skills and understanding of Next.js concepts.

## Purpose

- Practice Next.js features
- Experiment with new ideas
- Learn and improve coding skills

## Getting Started

1. Clone the repository.
2. Install dependencies:  
    ```bash
    npm install
    ```
3. Start the development server:  
    ```bash
    npm run dev
    ```
